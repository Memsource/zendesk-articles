New Features and Improvements

Approved Terms for Easier Identification

With the release of this feature, New terms will be marked in grey while the Approved terms will remain black. This is already available in the Web Editor. 

Until now, it was only possible to distinguish between New and Approved terms using the metadata details at the bottom of the CAT pane:

 

Marking Terms as Preferred and Having Preferred Terms Displayed First in CAT Pane

The Desktop Editor now supports marking terms as Preferred when creating a term base entry. Also, a preferred target term for a source term will be positioned above any other term associated with that source term in the CAT Pane.
