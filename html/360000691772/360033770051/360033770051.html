TYPO3 is a free and open-source web content management system written in PHP.

TYPO3, along with Drupal, Joomla! and WordPress, are popular content management systems.

By using a TYPO3 connector, users translate the page content of the TYPO3 CMS, e.g., standard pages. In a Memsource project, pages are manually selected for translation and then manually uploaded back to a TYPO3 site. Automated Project Creation can be configured to download/upload the files automatically.

Requirements

  • TYPO3 Version: 8.7 or higher
  • PHP Version: 7.0 or higher

Content from Typo3 is always imported as HTML files.

Set up the Memsource Connector in your TYPO3 CMS

  1. Download the Memsource Connector from TYPO3 extensions page and install it in Typo3 CMS.
  2. Enable the Memsource extension in TYPO3 Admin Tools > Extensions.
  3. Get the Memsource Connector Token in the TYPO System by clicking on Memsource. Debug mode and download logs are enabled on this page.
  4. Setup target Languages in the List section in TYPO3

Set up the TYPO3 Connector in Memsource

After plugin is set up in TYPO3, log in to your Memsource account and follow these steps:

  1. From the Memsource Setup page, click on Connectors under the Integrations section and select New.
  2. Select the TYPO3 option.
  3. Enter the URL of the TYPO3 CMS (for example http://memsourcedemo.com/typo3-v8) in the Host field and the Memsource Connector Token in the Token field.
  4. Enter the source language. 

    Note

    TYPO3 will not store this information
  5. Click Test connection.
    Memsource connects to the TYPO3 site and displays a list of target languages configured in the TYPO3 instance.
  6. Save the connector.
    The connector is added to the list on the Connectors page.
