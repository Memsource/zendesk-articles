New Features and Improvements

Approved Terms Visually Indicated

Until now, it was only possible to distinguish between New and Approved terms using the metadata details at the bottom of the CAT Pane. With the release of this feature, New terms will be marked in grey.

 

Marking Terms as Preferred

As in Memsource, it is now possible to mark terms as Preferred when creating them from within the Memsource Editor for Web. The preferred translations can be identified by their designation at the bottom of the CAT Panel (in the metadata section).

 

Fixed Bugs

  • Due to a bug in the web editor, only the first occurrence of a term was highlighted in each of the segments. This has now been fixed.
