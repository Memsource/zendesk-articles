v20.14.4 - July 21, 2020

New Features and Improvements

  • SSO feature, which redirects the user to the SSO website to log in, is now supported.

  • The sentence-case style capitalization is used for UI strings.

Fixed Bugs

  • Keyboard shortcuts Ctrl+K and Ctrl+Shift+K didn't work as expected in a job that was opened in the read-only mode. This has been fixed.

  • When pressing CTRL+K while in the target segment, direction of the concordance started with source language. This has been fixed.

  • Linguists were not able to use the filtering feature via Ctrl+Shift+F in a job that was opened in the read-only mode. This has been fixed.

  • When clicking into unfocused segment which was covered by either the bottom or upper panel, the segment was not scrolled to the center of the table. This has been fixed.

  • The Changes tab in the CAT pane didn't display changes as expected while switching between the Editor and other apps. This has been fixed.

  • Recently used symbols were cleared when the Memsource Editor for Desktop was updated to a new version. This has been fixed.

  • Auto-complete didn't work in case the filter for 1st repetition was enabled. This has been fixed.

v20.6.3 - March 31, 2020

New Features and Improvements

  • Segments and Comments can now be filtered by Created by and Last Modified by dates.

  • In-Context Preview presents the Target preview by default instead of the Source.

Fixed Bugs

  • Character count was not updating without moving to a different segment to trigger update. This has been fixed.

  • Unconfirming repetition segments could cause the word count status bar to indicate negative numbers. This has been fixed.

  • Confirming repetition segments was not being reflected in the word count status bar. This has been fixed.

  • Text losing position in a segment when a filter is cleared. This has been fixed.

  • Uploading MXLIFF to server fails if Linguists may edit source is changed during the translation. Linguist is now notified of why upload is not possible.

  • Reconfirming a first repetition unconfirms later repetitions. This has been fixed.

v20.5.3 - March 17, 2020

New Features and Improvements

  • A paper clip icon now indicates if a Reference file or note has been uploaded to a project.

  • Invalid user names and credentials can now be removed from login preferences.

Fixed Bugs

  • Oriya/Odia script not working in correctly. This has been fixed.

  • Auto-propagated repetitions are confirmed even if Set segment status to confirmed for repetitions is set to false. This has been fixed.

  • Machine Translation populates on CAT pane even when turned off (Admin and PM users). This has been fixed.

Fixed Bugs

v20.2.7 - February 19, 2020

New Features and Improvements

  • A new version of the Memsource Editor for Desktop has been released with general stability improvements.

Fixed Bugs

  • Every segment with the same source was considered a repetition regardless of the Context key (if the job was imported with Context key). This has been fixed and only segments with the same source text and the same Context key are marked as repetition in the Editor.

v20.2.6 - February 18, 2020

New Features and Improvements

  • Instant QA and Auto-complete items are now available in the Tools menu.

  • The first instance of a repetition is now specifically identified.

  • The section Created By / Modified By / Show Changes has been divided into three sections providing better control of filtering options.

Fixed Bugs

  • Use of Ctrl+Shift+Up/Down Arrow did not select text as expected. This has been fixed.

  • Text could not be selected from the Preview pane for copying elsewhere. This has been fixed.

  • Confirming segments in workflow steps did not affect the Word Count Status Bar. This has been fixed.

v20.1.2 - January 21, 2020

Note that as of this release the versioning strategy has changed. Version number will reflect the year of the release followed by the number of the release during the year. This release is the 1st of the year 2020.

Fixed Bugs

  • Right-clicking on the file file name in the Status bar does not provide an option to copy. This has been fixed.

  • Reconfirming segments results in incorrect calculation in status bar. This has been fixed.

  • Filtering for quoted words with regex selects all words between the quoted words. This has been fixed.
