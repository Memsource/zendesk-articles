v21.15.2 - April 28, 2021

Fixed Bugs

  • Custom tags are not propagated in repetitions if there are no tags in source. This has been fixed.

  • Terms added via the Add term function were not always saved to the term base. This has been fixed.

  • Content of CSV files wasn't being displayed properly in the in-context preview. This has been fixed.

v21.12.6 - April 13, 2021

Fixed Bugs

  • Safari: Double clicking a word does not select it in Mojave version 10.14.5, Safari 12.1.1. This has been fixed.

  • Scrolling stops when repeatedly pressing the PgUp/PgDn keys. This has been fixed.

  • Incorrect apostrophe marks applied for Catalan. This has been fixed.

  • Imported VTT files with newline in a segment are showing as merged. This has been fixed.

  • Entering a double byte character (Japanese) just after a tag (both opening tag of tag pair and single tag ), the first following character is deleted. This has been fixed.

  • Certain characters are displayed in a different font than usual in Simplified Chinese. This has been fixed.

  • A partial MXLIFF without any <m:tunit-target-metadata/> is being sent when tags were deleted from a target when it should be empty. This has been fixed.

  • Line breaks from a term's Note field in a term base are not being preserved. This has been fixed.

  • Joined jobs in RTL languages are presenting segments in the wrong order. This has been fixed.

v21.11.1 - March 30, 2021

New Features and Improvements

  • The names, warnings and info regarding terminology related QA checks have been changed to make them easier to understand.

Fixed Bugs

  • Some segments cannot be edited even if unlocked. This has been fixed.

v21.8.1 - March 16, 2021

New Features and Improvements

  • When filtering with Source/Target, Context key, Context note, and Tags marked as TRUE, these options are combined in the filtering results.

v21.4.2 - March 2, 2021

Fixed Bugs

  • Tags are being incorrectly overwritten with next tag and text. This has been fixed.

  • Highlighting is incorrect when filtering by emoji. This has been fixed.

  • Emojis in the end of a segment cannot be replaced. This has been fixed.

  • Find Next and Replace are not working correctly when working with emojis. This has been fixed.

v21.2.2 - February 16, 2021

Fixed Bugs

  • Copying/pasting source text preceded by a tag resulted in the tag being pasted into the target segment along with the first character of the selection. This has been fixed.

  • Some segments in a job are being left empty when progressing from a previous workflow step. This has been fixed.

v20.75.3 - January 19, 2021

Fixed Bugs

  • The cursor sticks on the Changes pane when confirming a segment. This has been fixed.

  • The editor freezes and requires a browser refresh to login again when adding to a TB. This has been fixed.

  • The word constructor cannot be used due to being a JavaScript keyword. This has been fixed.

  • Search results cannot be inserted into target by double-clicking or using keyboard shortcuts. This has been fixed.

  • The CTRL+arrow keyboard shortcut goes in the wrong direction in an RTL language. This has been fixed.

  • Incorrect quotation marks are used when typing in Dutch. This has been fixed.

  • Undoing results in joined jobs or locked segments results in incorrect display. This has been fixed.

  • Japanese characters are not being displayed correctly. This has been fixed.
