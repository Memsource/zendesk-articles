New Features and Improvements

Filtering Segments by Tag Content

If the tags are expanded in the editor, users can now also filter segments by the content of these tags. This allows Linguists or Project Managers to easily identify tags with problematic formatting.

Fixed Bugs

  • Repetition arrows were not greyed out if the Auto-propagate repetitions option was disabled in the project. This bug is now fixed. 
  • Ignoring one of the QA warnings for a given segment automatically ignored multiple warnings for the same segment. This issue has now been resolved.
