Improvements Made to the Jukkou MT Engine

Improvements have been made to the way Memsource interacts with the Jukkou MT Engine. These improvements ensure that translation requests for segments are not duplicated.

 

Fixed Bugs

  • When translating from an RTL language into an LTR language, spaces after the period would appear before the period in the target language. This issue has now been resolved.
