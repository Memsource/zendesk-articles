Here is a list of features specific to the Memsource Editor for the web. General tips and tricks for both Memsource Editors can be found in the Tips for Working in the Memsource Editors article.

Find and Replace

In order to activate the Replace feature, you actually need to filter for the contents in question. Type the word or phrase you want to replace into the Filter Target Text field. You will then see a new Replace section appear above the Segment Status section. (If this is not visible to you, click the arrow to the left of the Filter Source Text field to expand this section.) Then, type the word or phrase you want to replace the original text with, and select Replace or Replace All.

Note: The Replace option for the source language is not available in the Web Editor, only in the Desktop Editor.

Loading Large Jobs

Documents in the Web Editor are loaded in batches. Large jobs can take a while to be loaded completely, so you might experience some missing segments at end of your file. Every time you press PageDown or scroll down with your mouse, the page will load more segments—just give it a few seconds.
