File renaming automates the renaming of downloaded (translated) files.

To bulk rename downloaded files, follow these steps:

  1. From the Setup  page, scroll down to the Project Settings section and click on File Renaming.

    The File Renaming page opens.

  2. Apply required macro. Macro names are case-sensitive.

    Supported macros:

    • {path}

      Only for files imported in ZIP format with a folder structure.

    • {fileName}

      Characters can be removed from the original file name by using a character position (starting from 0) to select part of the name. Examples:

      • {fileName[2:6]} to keep the 3th to the 7th characters of the file name

      • {fileName[0:-3]} to cut off the last 2 characters.

    • {sourceLang}

      Adds source language

    • Use these macros to represent target language.

      • {targetLang}

      • {targetLangName}

      • {targetLangRFC}

        • Examples: es, en-US, az-Latn-AZ

        • Based on RFC 5646

        • To programmatically convert Memsource languages codes to RFC:

          1. Replace _ with -.

          2. Convert with the regexp (in PCRE format): ^([a-z]{2,})(?:(-[a-z])([a-z]+))?-(?:([a-z]{2})|([a-z])([a-z]{2,})|([0-9]{3}))$

        • Conversion example.

      • {targetLangAndroid}

        • Examples: es, en-rUS

        • To derive the tags programmatically, convert with the regexp (in PCRE format): -([A-Z]{2})$

        • Conversion example.

      • {targetLangAndroidBCP}

        • Examples: b+es, b+en+US

        • To derive the tags programmatically:

          1. Prefix the RFC regexp with b+.

          2. Replace - with +.

    • {workflow}

    • {status}

    • {date}

      Date of file download in the PM's timezone

    • {time}

      Time of file download in the PM's timezone

    • {userName}

      Adds username of the assigned Linguist or Vendor. If multiple Linguists are assigned, the name will be empty.

    • {replaceExt[xx]}

      The file extension is replaced by that specified within [ ] brackets.

      • Example:

        If {replaceExt[txt]} is specified, the file test.POT is downloaded as test.txt.

      • Changing extensions does not change the format itself; renaming 'docx' to 'xlsx' does not change a text file into a spreadsheet file.

  3. Click Save.

    Specified macros are applied to downloaded files.
