Introduction

Microsoft Translator is one of the general machine translation services that are immediately available in Memsource Cloud. In addition, users can create custom machine translation engines (e.g. for specific domains) that are trained with their own data via Microsoft Translator Hub, which is now directly supported in Memsource Cloud. Microsoft Translator with Feedback (free) is no longer available.
