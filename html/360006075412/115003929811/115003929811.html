Active languages

When you select source and target languages, you are only offered languages that are set in the system as "active". Languages can be activated and deactivated under Setup by going to the Administration section and clicking on Active Languages.

This setting will affect all language selections including Automation widgets and Automated project creation.

.

Spellcheck

Memsource offers an embedded spellchecker for most of the major languages available in Memsource (marked in the language table above). 

For languages that we do not support directly, we recommended using the spellchecker in an external application that supports DOCX files such as Microsoft Word, Open Office Writer, or Livre Office's Write. See more details in our dedicated Spellchecker article.
