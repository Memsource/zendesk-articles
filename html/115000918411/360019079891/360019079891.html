Create a New Subdomain

action: api/v2/subDomain/create
    token                       string
    name                        string                             

response: JSON
    {
      "id": 238                 // id of the new subdomain
    }

Get a Subdomain

action: api/v2/subDomain/get
    token                       string
    subDomain                   domain(SubDomain)

response: JSON
   {
     "id":3,
     "name":"subDomain"
   }

List Subdomains

action: api/v2/subDomain/list
    token                       string
    page                        integer            O(0)
    name                        string             O

response: JSON
   [
     {
       "id":3,
       "name":"subDomain"
     }
   ]

Edit a Subdomain

action: api/v2/subDomain/edit
    token                       string
    subDomain                   domain(SubDomain)
    name                        string

response: empty

Delete a Subdomain

action: api/v2/subDomain/delete
    token                       string
    subDomain                   domain(SubDomain)

response: empty
