v21.5.3 - March 16, 2021

New Features and Improvements

  • This release includes general optimization improvements.

v21.5.1 - March 2, 2021

Fixed Bugs

  • In-context preview unexpectedly expands to fill the window. This has been fixed.

v21.3.3 - February 24, 2021

Fixed Bugs

  • The Changes Pane is showing incorrect results. This has been fixed.

v21.3.2 - February 16, 2021

New Features and Improvements

  • The Ctrl+Shift+Tab keyboard shortcut can now be used to switch between side panes in the reverse direction.

Fixed Bugs

  • Tag metadata from TMs is being ignored. This has been fixed.

  • Filter setting options are not maintained for an instance after clearing the filter. This has been fixed.

  • The width of the CAT pane is not being consistently saved. This has been fixed.

v21.2.3 - February 2, 2021

New Features and Improvements

  • macOS users can use press-and-hold on the keyboard to bring up a list of accented characters.

  • Project Managers can now prevent Linguists from editing the content of tags.

v21.2.1 - January 19, 2021

Fixed Bugs

  • Newline character \r is being changed to \n on import. This has been fixed.

  • Linguists can confirm locked segments without using F2. This has been fixed.

  • Using a quotation mark in a source/filter field causes the editor to crash. This has been fixed.

v20.26.2 - January 6, 2021

Fixed Bugs

  • Created and Modified metadata is not correctly displaying the time stamp when changing the language to Russian. This has been fixed.

  • Selecting multiple QA warnings is not always working. This has been fixed.
