In this article you will find a list of API calls or parameters which are not supported and some solutions or workarounds.
 

  • API Call returns Web API returns AuthApiDisabled
    This happens when your Memsource edition does not support API calls. The only edition that does not support API calls is the Personal edition. 
     
  • Get file by name
    Unfortunately this is not possible - the file can be uploaded to several worksteps so it would be impossible to identify the right one. Only jobPart is supported in GetJob API.
     
  • Create or Edit  WorkflowStep
    this call is not available. The work flow steps need to be created and edited manually in Cloud. That includes order of WorkFlow steps. Only List is available via API.
     
  • API for deleting multiple domains
    This API is not available.
     
  • Assign linguist to multiple task in one call
    not available. You have to call each task in EditJob API individually.
     
  • File size limit in createJob
    File size limit for API is the same as for Cloud: 1GB per file or 200MB for zip.
     
  • Delete multiple projects
    It is not possible to delete multiple project in one API call. Delete project must be called individually.
