New Features and Improvements

Preferred Terms Displayed First

A preferred target term for a source term will be positioned above any other term associated with that source term in the CAT Pane.

This builds on the Preferred Terms feature released a couple of weeks ago that allows terms to be marked as Preferred. 

 

Selecting All Before/After the Cursor

The list of supported commands and keyboard shortcuts for the web editor has been extended. It is now possible to select everything before or after the cursor.

Windows/Linux users can use CTRL+SHIFT+HOME/END to jump to the beginning or end of the text after the cursor. Mac users can use CMD+SHIFT+UP/DOWN arrows to achieve the same result.
