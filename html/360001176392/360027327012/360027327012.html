New Features and Improvements

Support for Term Level Metadata

To allow more complex work with terminology in Memsource, we are making it possible for users to specify advanced metadata fields for each of the terms.

The following metadata are currently supported for new or already existing terms:

  • Part of Speech
  • Gender
  • Number
  • Note
  • Short Translation
  • Term Type

If you would like to learn more, you will be able to find all the necessary information in our documentation. Please note the information will be published at the time of the release.

 

Term Base Redesign

In order to fully accommodate to the changes being introduced for term metadata, we have also redesigned the UI of the Term Bases to provide this information more easily.

The individual translations of your terms are now displayed in rows and columns have been rearranged to allow displaying of the various available term metadata.

You can also decide which languages are you interested in seeing as well as filter or show/hide respective metadata columns.
