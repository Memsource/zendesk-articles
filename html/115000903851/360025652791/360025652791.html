Memsource Translate (now in beta) evaluates the quality of different machine translation engines and automatically selects the best performing engine for a project, based on the language pairs used. It supports 69 language pairs and also includes Machine Translation Quality Estimation (MTQE). Below, you'll find the answers to some commonly asked questions about this new feature.

 

How do I Buy Memsource Translate Characters?

After you use up your free Memsource Translate characters, you'll have to purchase new ones to continue using this feature. You can do this by going to your account’s Settings and clicking on Machine Translation Engines in the Integrations section. In the Memsource Translate section, you'll see a button that gives you the option to Top Up Characters.  For a breakdown of our pricing for these character bundles, please see the chart below.

How Many Free Memsource Translate Characters do I Receive?

Every account will automatically start out with free Memsource Translate characters:

  • Personal edition: 50k characters
  • Team Start edition: 250k characters
  • Team and Academic editions: 500k characters
  • Ultimate, Enterprise, and Enterprise starter editions: 2 million characters

 

Which Machine Translation Engines does Memsource Translate Support?

Currently, Microsoft Translator and Google Translate are available with more engines to follow. Please note: Memsource Translate does not support custom MT engines, and it is not possible to create multiple instances of Memsource Translate at this time.

 

Do I Have to Buy MTQE Characters Separately?

No, MTQE is available with Memsource Translate at no extra charge.

 

What will Happen with my Existing Free/Purchased Characters? Will They Get Lost?

If you still have free Microsoft Translate characters in your account, you'll be able to use them for up to four months.

 

Is There Any Chance to Get Monthly Free Characters from Microsoft?

You can still use Microsoft Translator and its free characters. To do this, however, you'll need to disable Memsource Translate. You'll also need to set up an account with Microsoft and obtain an Azure Key to complete this. (Ignore the Develop your own app heading.)

—Alternatively—

Unfortunately, it is no longer possible to get free Microsoft Translator characters.
