To archive a project, follow these steps:

  1. From the Projects page, select the project to be archived and click Archive.

  2. Verify that the project is to be archived.

  3. The project is moved to the Archived Projects page.

To restore a project, follow these steps:

  1. From the Setup  page, scroll down to the Tools section and click on Archived Projects.

    The Archived Projects page opens.

  2. Select the project(s) to be restored.

  3. Click Restore.

  4. Verify the restoration.

  5. The selected project(s) are returned to the Projects page.
