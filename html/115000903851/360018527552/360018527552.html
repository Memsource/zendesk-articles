A Linguist portal is a web page, where the linguists can manage all their translation jobs assigned by the project managers.

They can see all the jobs assigned to them by the Project managers, they can also see the projects, reference files, Translation memories, and Term bases (if their user's rights allow it).

Linguists can accept or decline the jobs, download the files for translating in the Memsource Desktop Editor or open it directly in the web browser in the Memsource Web Editor. Once the translation is done, the linguist can upload the file back to the server (if working offline before) and set the job as Completed.

Sign In

The Linguist Portal is accessible only with a special Username and password.

  1. If you are a new user and received a Username created for you by the Project manager (with no password), please follow an article Create / reset password
  2. If you already have the username and the password, Sign in on www.cloud.memsource.com (your project manager can provide you with a custom URL link instead).
    Important: Please do not use the "Sign In with Google"

 

Main Page

  • The Main menu in the left-hand column offers listing of the projects based on the status of the jobs assigned to the linguist.
    • New: All projects with new jobs assigned to you by the project manager. If a project has some jobs New and some jobs Accepted or Completed, it will be listed on this page.
    • Accepted: Projects where all jobs assigned to you are Accepted by you. Projects where some jobs are Accepted and some Completed, will be also listed on this page
    • Completed: Projects where all jobs assigned to you are Completed.
  • RECENT ITEMS menu on the left-hand side will offer a list of the items (projects, jobs, etc.) you have recently opened.
  • Filter
  • Customize columns - It is possible to customize the columns shown in the Project list. Click on the "Settings" icon on the right-hand side of the first row and then click on "Customize columns".

    You can select: #, Name, Created, Domain, Subdomain, Owner, Source, Target Languages.
  • Every time a new version of the Memsource Editor for Desktop is relaesed, you will see a message at the top of the main page.
  • You can also see a notification about our Memsource Mobile application for iOS and Android.
