New Features and Improvements

  • Instant QA and Auto-complete items are now available in the Tools menu.
  • The first instance of a repetition is now specifically identified. 
  • The section Created By / Modified By / Show Changes has been divided into three sections providing better control of filtering options.

Fixed Bugs

  • Use of Ctrl+Shift+Up/Down Arrow did not select text as expected. This has been fixed.
  • Text could not be selected from the Preview pane for copying elsewhere. This has been fixed.
  • Confirming segments in workflow steps did not affect the Word Count Status Bar. This has been fixed.
