New Features and Improvements

Terminology Between CSV files and LOG Files is Now Consistent

Previously, there was an inconsistency in the terminology used in analyses downloaded as CSV or CSV with chars files compared to those downloaded as LOG files. This has now been changed so that CSV file types use the same terminology as LOG file types.

Only two parameters needed to be changed:

  • Context TM is now Context Match
  • 100% Matches is now 100%

Important: This change may have an effect on the way some applications process the CSV (Integration with Memsource). These applications must be updated accordingly.

Fixed Bugs

  • Due to recent changes made to the GitLab connector, GitLab files from subfolders were not always able to be imported for translation. This bug has now been fixed.
