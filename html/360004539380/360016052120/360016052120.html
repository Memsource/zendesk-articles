v21.5 - March 23, 2021

New Features and Improvements

  • API: Up to 1,000 jobs or 300,000 segments can now be downloaded via POST /api2/v1/projects/{projectUid}/jobs/bilingualFile endpoint.

  • Contentstack: Management token can now be used for authentication.

Fixed Bugs

  • API: headerFooter parameter in doc file is not working. This has been fixed.

  • AEM: Images in CSS files are not having URL redirection applied. This has been fixed.

v21.4 - March 9, 2021

New Features and Improvements

  • Magento 1.0.7: Now supports Magento 2.4 CE.

  • WordPress: Connector now supports basic authentication for WordPress installations secured by it.

Fixed Bugs

  • COTI: Projects are not being created if .XML does not contain archive attribute. This has been fixed.

  • Contentful: Extra line breaks (\n) are being added to exported content. This has been fixed.

v21.3 - February 23, 2021

New Features and Improvements

  • Figma: Management of UI strings for translation and ability to resubmit to an existing project.

  • AEM: In-context preview is available for shared projects and jobs.

  • Drupal: Plugin in now compatible with Drupal 9 and PHP 8.

    The Job Item label is also now part of the filename.

    After a job is pulled by Drupal, the status in Memsource can now be changed to Delivered.

    All target languages can now be submitted into a single Memsource project.

  • Contentstack: Assets can now be translated.

  • API: API Token now accepted in URLs.

v21.2 - February 9, 2021

New Features and Improvements

  • Salesforce Knowledge: There is now the ability to specify publish status in the connector configuration.

  • Contentstack: Content authors can pass the list of target languages to Memsource via a dedicated field.

  • Zendesk Connector: Any Zendesk Guide language, not only the default language, can be used as the source language.

Fixed Bugs

  • Zendesk Connector: 400 error when exporting jobs back to Zendesk. This has been fixed.

  • Magento Connector: Not working properly with all types of SSL certificates. This has been fixed.

v21.1 - January 26, 2021

Fixed Bugs

  • Figma: File import settings aren't taken from the template. This has been fixed.

  • Git: Export to git handles merge conflicts and retries up to 5 times when an error is detected.

v20.26 - January 12, 2021

New Features and Improvements

  • Contentstack: APC now supports CMS workflows.

  • Automated Project Creation using the Git connector: When a relative path is used (ie: not starting with /) in the Target Subfolder Path, it is interpreted as relative to the file (not the monitored folder).

Fixed Bugs

  • Sketch plugin: Symbols pages needs to be submitted for translation separately in order for the text in the Final page to appear translated. This has been fixed.

v20.25 - January 5, 2021

Fixed Bugs

  • COTI: APC is not working. This has been fixed.
