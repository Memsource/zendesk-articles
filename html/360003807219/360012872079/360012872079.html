The Search Pane provides results from searches of Translation Memories and Term Bases. This is also known as a Concordance search. Wildcard search is supported.

The Search Pane can be accessed via:

  • The Tools menu.

  • Selecting the Search pane.

  • Selecting a string and using the keyboard shortcut CTRL+K for fuzzy search or CTRL+SHIFT+K for exact search.

Click on a search result to present the text in the lower pane.
