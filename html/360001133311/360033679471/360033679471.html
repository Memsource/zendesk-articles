New Features and Improvements

Arial Font is Used for Arabic

Arial font is now the primary font used for Arabic in the editor.

 

Fixed Bugs

  • Due to a bug, when working with RTL languages, the cursor position and highlights would sometimes be shifted incorrectly to the right. This has now been resolved.
  • When viewing the Changes tab and then confirming a segment, the editor would sometimes switch to the CAT tab. This has been fixed.
