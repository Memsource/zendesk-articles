New Features and Improvements

Adding a Single TM in Read-Only Mode

It is now possible to add a single TM in Read-only mode to a project, without needing to also have a TM in Write mode. The single Read-only TM can be added for all Workflow steps or for just some of these steps.

 

Reference Files Section Added to the Automation Widget 

There is now a Reference Files section in Automation Widgets. This allows users to submit reference files more easily and with less pre-processing.

 

Extracting Marketo My Tokens

When using Marketo, "My" Tokens are now displayed in the position where they occur in the document as an inline tag.
