Human Translation Engines are hybrid AI/Human translation services. Memsource can use these platforms in the same manner as Machine Translation Engines.

Supported Engines:

  • Gengo

  • Unbabel

Set up Human Translation Engine

To set up a Human Translation Engine, follow these steps:

  1. From the SetupÂ  page, scroll down to the Integrations section and click on Human Translation Engines.

    The Human Translation Engines page opens.

  2. Click Create.

    The Create Human Translation Engine window opens.

  3. Select an engine from the dropdown list and click Create.

  4. Fill in the required fields and select options.

  5. Click Save.

    The selected engine is added to the list.

See Gengo or Unbabel documentation for more help.
