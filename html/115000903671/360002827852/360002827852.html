With "Assign Providers from a Template" feature in the Project > Tools, a Project Manager can assign providers (Linguists or Vendors) to each of the project workflow steps using a pre-existing Project Template

It is possible to assign providers in different ways:

a) Assign all providers to all jobs at once - don’t select any jobs and navigate to the Assign Providers from a Template option in the Tools menu.

b) Assign some providers to some jobs across all workflows - don’t select any jobs and navigate to the Assign Providers from a Template option in the Tools menu.

c) Assign providers to selected jobs only - select the jobs first and then navigate to the Assign Providers from a Template option in the Tools menu.

Additional information: Assigning providers from a template won’t overwrite any existing assignments. You can assign providers from multiple templates as long as the templates match the criteria.

Once the Assign Providers from a Template option is selected, a pop-up appears with a drop-down list of Project Templates. Select the appropriate template.

It is important to note that the number of workflow steps and the source language in your chosen project template must match those in your new project, if not the project template will not appear in the list.  The target languages can be different, but at least one of the target languages must be the same. If anything needs to be changed in a template it can be edited by selecting Project Template from the main settings and clicking on the required template. The Edit Project Template page will open.
