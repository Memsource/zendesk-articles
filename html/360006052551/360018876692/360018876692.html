TXT import options:

  • Translatable text: Users can specify (using Regexp) what lines should be imported.
  • Convert to Memsource tags: Matching text will be converted to Memsource tags using Regexp. 

Note: If, for example, you want to use regular expression to extract only the value of content attributes from one line <meta name="breadcrumbs" content="Cheap cars"> the regexp should be: (?<=\<meta name="breadcrumbs" content=")[^\"]+
