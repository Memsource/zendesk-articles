A translation memory (TM) is a database of previously translated text and is a key aspect of every CAT tool. The text is split into smaller segments (usually sentences or titles) in a process called segmentation. The original segment and its translation are then saved into the translation memory as a "translation unit". The TM can have several target languages (though only one source language) and can be used in multiple projects simultaneously.

The Benefits of Using TMs

There are two main benefits to using TMs. The first is that TMs allow translators to reuse translations, which speeds up the translation process and reduces costs. The second is that TMs help to ensure that translations are consistent. This is especially useful when, for example, different translators are working on the same project for a specific client.

It's important to note that while the use of translation memories is highly recommended, there are some limitations. While there are no specific limits to how many target languages the TM can have or how many translation units can be saved in the TM, please bear in mind that very large TMs (with millions of segments) can slow down the performance of searches, pre-translation or analysis and can be difficult to maintain and edit. There is also a limit to the file size (1 GB) for exporting and importing. A project can have multiple TMs, so it is better to have a few smaller TMs then a very large one.

Setting Up a Translation Memory

There are a number of things you can do with a translation memory, but first, you have to set one up. Once you have that done, you can:

  • Assign the memory to a Memsource project so that the ongoing translation will be saved there.
  • Import a TMX from another CAT tool or a special Excel file into your Memsource TM.
  • Align previously translated text with its original version and import this into your Memsource TM.
  • Edit the parameters of the TM.
  • Search and edit the content of the TM directly in Memsource.
  • Export it and edit outside of the Memsource Editor, or import it into another CAT tool.

Using a Translation Memory

After your translation memory is set up, it's time to put it to use. The most common uses for a translation memory are:

  • Analysis of jobs to see if there are any matches between your job and your TM.
  • To pre-translate your job in a batch.
  • To display matches from the TM to your Linguists or Vendors in the Memsource Editor during the translation process.
