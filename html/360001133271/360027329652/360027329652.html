New Features and Improvements

Term Level Metadata in Memsource Editor for Desktop

With the introduction of additional term metadata, the way of displaying the metadata in the Memsource Editor is modified accordingly in order to provide the same information needed for localization. The design of metadata for TM and MT changes as well.

The Note and Usage fields have been removed from the Add Term window in order to centralize all the metadata manipulation into the cloud interface where all the other information can be specified as well.
