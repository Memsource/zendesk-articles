Setting up Custom Domains and/or HTTPS

To setup a Custom Domain and/or apply HTTPS, follow these steps:

  1. Purchase a domain name (cloud.abctranslations.com).
  2. Create a CNAME DNS record for cloud.abctranslations.com pointing to cloud.memsource.com 
    Use, for example, ping.eu to ensure settings are correct. The change usually takes approximately 24 hours.
  3. Purchase an SSL certificate for the domain.
    • HTTPS is provided using Server Name Indication (SNI) and supports these browsers.
    • Generate a Certificate Signing Request (CSR) when purchasing the SSL certificate.
  4. Send the domain name to the Memsource Support team. Include files with the extensions .cer, .key or .pem containing SSL Certificates and private keys.
