Fixed Bugs

  • White space characters at the end of lines for CJK languages were not correctly displayed as non-printable characters. This issue has now been fixed. 
  • If the Copy Source to Target option was selected in the Preferences of the Desktop Editor, the source was not correctly inserted into the target segment. This bug has now been fixed. 
  • It was not previously possible to insert only part of the source using the Copy Source to Target option in the Desktop Editor. This has been fixed in the latest release.
  • When a locked segment contained a spelling warning, Linguists were able to alter locked segments using the spell checker suggestions. This has now been fixed.
