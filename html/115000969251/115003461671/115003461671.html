If you want to select multiple jobs that appear on different pages, the solution is to use the filter feature that is available in the Jobs section.

For instance, if you need to analyze files that are assigned to a specific Linguist, you can do this first by filtering the files so that only ones assigned to that Linguist are shown, and then launching the analysis of these files.

If you still get more than one page of jobs after applying the filter, follow these steps:

Step 1: Select all of the jobs on the first page of the filtered results. (You can do this by selecting the checkbox at the top.)

Step 2: Once you do this, a new checkbox will appear called Select all jobs. This will allow you to select all filtered jobs, even though they are across multiple pages.

Step 3: Launch the analysis.
