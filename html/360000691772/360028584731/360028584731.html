Microsoft Azure is the part of Azure Storage that can be integrated with Memsource. It is a cloud-based platform that provides serverless containers to help you build, manage, and deploy applications. Below, you’ll find instructions on how to integrate your Azure account with Memsource.

Setting up the Connector

Before setting up the connector in Memsource, log in to your Microsoft Azure account through the Azure portal. Click on Storage accounts in the left-hand navigation panel. Select an existing storage account or create a new one. In the Settings menu, click Access keys and copy either key 1 or key 2. You’ll need this later in Memsource.

Once you have the key copied, go to Memsource. Navigate to Setup and scroll down to the Integrations section. Click on Connectors. After clicking on New, you can choose the connector you wish to create. Give it a name and change the Type to Microsoft Azure.

Getting your Connection String

The final field you’ll need to fill out is the Connection String field. This will be one long string with the key you copied from Microsoft Azure in the middle. Replace the ✱ in the following string with that key:

DefaultEndpointsProtocol=https;AccountName=memsourceconnector;AccountKey=✱==;EndpointSuffix=core.windows.net

Then, paste the entire string into the Connection string field. Click on Test Connection. If you’ve done everything correctly, a little check mark will appear next to the Test Connection button. Click Save.
