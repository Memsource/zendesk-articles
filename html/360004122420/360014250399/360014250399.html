Price lists are used for calculating Quotes used in Automation Widgets.

In Ultimate and Enterprise editions, Quotes can be created manually from Analyses with Price Lists being available for selection.

Editable Default Price Lists for EUR and USD are pre-populated on the Price Lists page.

Create a Price List

To create a price list, follow these steps:

  1. From the Setup  page, scroll down to the Financials section and click on Price Lists.

    The Price Lists page opens.

  2. Click New.

    The Create Price List page opens.

  3. Provide a Name, Currency, Billing Unit and whether this will be the Default price list.

  4. Click Save.

    The new price list is saved and the Price List table is displayed.

Edit Price List and Add Language Pairs

Price lists can be added, deleted, cloned, imported and exported (XLSX) from the Edit Price List page.

To edit a price list and add language pairs, follow these steps:

  1. Create a new price list or click on the name of a list on the Price Lists page.

    The Price List table is displayed in either the Create Price List or the Edit Price List page.

  2. Click Add Language Pair.

    The Source Languages and Target Languages selection boxes are displayed.

  3. Select language pairings and click Add.

    The pairings are displayed in the Price List table.

  4. Apply pricing.

    Values added are automatically saved to the Price List.
