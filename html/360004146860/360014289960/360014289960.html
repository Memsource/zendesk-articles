PHP is not a standard translatable format and it is not fully supported by Memsource. Only text variables and arrays with specifically defined keys are imported for translation.

Some PHP files can be imported as HTML files by changing the File Type to HTML, or as TXT files using the Translatable text option with regexp.

File Types

  • .PHP

Import Options

  • Use HTML subfilter

    Imports HTML tags contained in the file. Tags can then be used with HTML File Import Settings.

  • Convert to Memsource tags

    Apply regular expressions to convert specified text to tags.
