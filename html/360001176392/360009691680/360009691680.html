New Features and Improvements

Yandex MT Engine Now Available

The Yandex MT engine has now been added to our list of MT engines, giving our users more options for use in their translations.

Support Added for HubSpot Blogs

The Memsource connector is now able to browse blog content in HubSpot and import that content as a job.

Support Added for Contentful Entry-level Connector.

The Memsource connector is now able to extract translatable text from composable entries.

"Linguistics may not set jobs with QA warnings to Completed" Moved in List
In order to make the list of QA warnings more streamlined, the QA warning Linguists may not set jobs with QA warnings to Completed has been moved to the bottom of the QA checklist.

Confirmation When Deleting Terms

When deleting terms from a term base, there is now a popup confirmation. This will help prevent users from deleting terms unintentionally.
