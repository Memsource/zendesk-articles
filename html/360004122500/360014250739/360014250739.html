To setup a Custom Domain and/or apply HTTPS, follow these steps:

  1. Purchase a domain name.

    Example:

    cloud.abctranslations.com

  2. Create a CNAME DNS record for cloud.abctranslations.com pointing to cloud.memsource.com.

  3. Use Ping to verify the settings are correct.

    DNS changes typically take 24 hours.

  4. Purchase an SSL certificate for the domain.

    • HTTPS is provided using Server Name Indication (SNI) and supports these browsers.

    • Generate a Certificate Signing Request (CSR) when purchasing the SSL certificate.

  5. Send the domain name to the Memsource Support team. Include files with the extensions .cer, .key or .pem containing SSL Certificates and private keys.
