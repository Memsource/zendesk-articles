Limits may be applied to YAML files.

File Types

  • .YAML

  • .YML

Import Options

  • Use HTML subfilter

    Imports HTML tags contained in the file. Tags can then be used with HTML File Import Settings.

  • Convert to Memsource tags

    Apply regular expressions to converting specified text to tags.

  • Import specific keys only (use regexp)

  • Export specific values (use regexp): See our Regexp article for more details.

  • Context key

    Specify a context key is saved with the segment to the Translation Memory and used for match context.

  • Context note

    Matching content is displayed in context notes.
