New features and Improvements

Sorting QA Warnings

In order to help better organize QA warnings, users can now sort them in ascending or descending alphabetical order.

Memsource Editor Icon Color Change

The color of the Memsource Editor for Web icon in Linguist Recent Items and the color of the Memsource Editor for Web favicon in the browser tab have been changed to black. This helps distinguish the Memsource tabs from the Memsource Editor for Web tabs.

 

Fixed Bugs

  • Due to a bug, confirming a segment while Instant QA was on would not always confirm that segment. This has now been resolved.
