New Features and Improvements

Linguists now Distinguished from Providers

Linguists and Providers are now distinguished from each other in the Memsource UI. Previously, Memsource would ask users to select Linguists to be assigned to a job. However, this did not take into consideration that other vendors might be assigned to a job. The new UI distinguishes these two roles to help users to select the most appropriate person for a job.

 

Adding Files from an Online Repository in Automation Widgets

When an automation widget has connectors assigned, an Add from Online Repository button now appears. The button will only appear if the widget has connectors that are assigned.

 

Showing Quotes to the Assigned Providers

Providers can now see how much they will get for a job in the Quotes section. Project Managers can assign the quote to a provider using the drop-down menu.

 

New Connector—Contentful

Memsource now supports Contentful as one of our connectors. You can establish a connection to it directly via Memsource and translate the following types of content:

  • Spaces
  • Environments
  • Entries

The connector supports both manual projects and Automated Project Creation.

Fixed Bugs

  • Occasionally, jobs would not finish loading when uploading a zip file after a project's creation. This has now been resolved.
  • When multiple files were imported into a single project that had no workflow steps, the total price displayed would be correct, but sometimes the prices for the individual files would not be. This has now been resolved.
