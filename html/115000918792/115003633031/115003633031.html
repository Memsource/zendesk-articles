Find and Replace

Find

To find words in the source or target, use the Text Filter fields. To learn more about text filters in the Editors, see the text filter article for the Desktop Editor. 

 

Note: Exact searches (using quotes) are not available in the Web Editor (only in the Desktop Editor).

Replace

Note: The Replace option for the source language is not available in the Web Editor, only in the Desktop Editor.

 

To Replace or Replace All, proceed as follows:

  1. Enter the words you need to replace in the text filter.
  2. Expand the Additional Segment Filter options on the left, and then expand the Replace menu.
  3. Type the word or phrase you want to replace the original text with, and select Replace or Replace All.
