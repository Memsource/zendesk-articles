Memsource provides five different roles that user accounts can be attributed to and two roles that functions as organizations for the sharing of resources. The Personal edition is restricted to a single user with Administrator rights.

Individual account types:

  • Administrator

  • Project Manager

  • Linguist

  • Guest

  • Submitter

Organizations:

  • Buyer

  • Vendor

Users Page

The User Page is accessed through either the left-hand navigation panel or clicking Users under the Administration menu in Setup .

From the Users page, Administrators and Project Managers can:

  • Create or delete users.

  • Bulk import new users.

  • Edit user details, role, rights, set relevancy and other options.

  • Email login instructions.

  • View login history (last three months).

User Licensing

With the exception of the Personal edition, the monthly subscription is based on the number of licensed users with Linguist users being provided in a package with a Project Manager license. The Team and Ultimate editions, for example, come with 10 Linguist accounts per Project Manager/Administrator. The Enterprise edition comes with an unlimited amount of users of all types.

An unlimited number of user accounts can be created with the Team Start, Team, Ultimate, and Enterprise editions, but there is a licensing limitation (the number of purchased Project Manager user accounts). Licensing limitations are based on the number of active users in the organization's Memsource account at any given time.

Example:

If an organization subscribes to 5 Project Manager accounts, they can create any number of PM accounts, but only 5 PM users can be active at any given time. The same principal applies to Linguist users.

With the Team, Ultimate, and Enterprise editions, the activation/deactivation of user accounts happens automatically. If the limit of licensed users is reached for a particular role, no more users of that type will be able to log in and use Memsource. Administrators and Project Manager can control which users are active and which are inactive from the User page.

Team Start edition does not include automatic user activation/deactivation. User status is managed manual from the User page and only active users will be able to log in to Memsource.

Team Start edition is intended for stable translation teams while Team, Ultimate and Enterprise editions are more appropriate for larger organizations with more dynamic teams.

Create a User

Usernames

Usernames must always be unique to not only the organization but to the whole of Memsource. Do not use a deleted username for creating a new user.

It is recommended to prefix usernames to ensure their uniqueness.

Example:

J. Doe of ABC Organization = ABC_j.doe.

J.Doe of DEF Company = DEF_j.doe.

To create a user, follow these steps:

  1. Users can be created from two places:

    • Click the Users in the panel menu.

    • Click New from the Users page.

  2. Provide user information, Username, Role, Time Zone, role based options and Relevancy if required.

  3. Click Create.

    New user is added to the list on the User page.

Bulk Import Users

To create several users at the same time, an XLSX table containing user data can be imported. A pre-formatted table is available to download to help with the import.

To bulk import users, follow these steps:

  1. From the Users page, click Import.

    The Import Users window opens.

  2. Click sample XLSX file.

    The pre-formatted XLSX file is downloaded to your system.

  3. Open the XLSX file in a spreadsheet editor.

  4. Enter the following user information if available (if anything but the Username is not immediately available, it can be added manually):

    • First name

    • Last name

    • Email

    • Username

    • Password

    • Role

    • Receive newsletter

    • Note

  5. Save the XLSX file.

  6. From the Import Users window, drag&drop the file into the window or click Choose File and select the file.

  7. Click Import.

    The users specified in the XLSX file are added to the Users page.
