If you are new to Memsource and are a translator, reviewer, or proofreader then this Getting Started Guide is for you. We also have a dedicated Getting Started Guide for Project Manager Users.

This Getting Started Guide for Linguists will take you through everything you need to know to get up and running in Memsource. It includes step-by-step instructions and concise demo videos so you will find your way around in no time.

Before you get going, just a quick note about the structure of Memsource.

Our tool comes in three parts:

  1. Memsource UI: A web-based environment where Project Managers create and maintain projects, and where Linguist users can view and accept jobs.
  2. Memsource Mobile App: From the app, you can review and accept translation jobs on the go. Download it for free for Android and iOS devices.  
  3. Memsource Editor: This is where the actual translations are done. The desktop editor is available free for download and the web editor can be accessed online.

 

This guide will focus on the Linguist Portal in Memsource, along with Memsource Editor. The Getting Started Guide for Project Managers provides an overview of managing projects in Memsource.
