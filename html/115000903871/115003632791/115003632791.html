You can add your own custom shortcuts to the keyboard shortcuts in the Desktop Editor. The Customize Shortcuts option is available in the Tools menu of the Desktop Editor. You can customize all the default shortcuts for commands in the File, Edit, Tools, and Format menus.

 

To change a shortcut:

  1. Select the Customize Shortcuts option in the Tools menu. The Customize Keyboard Shortcuts window will then appear.
  2. Search for the appropriate action or keyboard shortcut.
  3. Click on the shortcut button. The words Press Keys will appear.
  4. Type the new shortcut.
  5. Click Save.
