New Features and Improvements

New Languages Added to Memsource

Memsource now supports 21 additional languages, including languages from Myanmar, Cameroon, Angola, Kenya, and the Philippines. For a complete list of our supported languages, please see our Languages Supported article.

 

Improvement to Quote Update and Quote Approval Process
Now, Project Managers can send a notification directly from Memsource and ask Submitters to approve or decline updated quotes. Project Managers are then notified of the Submitters’ decision. Is also possible to apply the same process to provider quotes created manually in Memsource.

 

Memsource Now Using Google's Cloud Translation v3

In order to continue supporting Google's AutoML engine, Memsource will now be using Cloud Translation v3. This will help to provide additional security for you and your clients.

 

Fixed Bugs

  • After a PM deleted some of the relevancy data, the relevancy section would sometimes fail to load in Linguists' accounts. This has now been resolved.
