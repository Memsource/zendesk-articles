Memsource automation is based on customized templates set for service types, project types, and due date schemes. Texts submitted through either the Automation Widget or Automated Project Creation have these templates applied to them to generate pre-configured projects. This removes the need for project managers to manually define each project.

To prepare for Automation, follow these steps:

  1. Create a Service.

  2. Create a Project Template.

  3. Create a Due Date Scheme.

  4. Set up Automated Project Creation and/or the Automation Widget.
