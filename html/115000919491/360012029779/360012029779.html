Analyses can be created and viewed in Memsource Mobile.

To create an analysis, follow these steps:

  1. From a Project page, tap on Analyses.
    The Analyses page opens.
  2. Tap the + icon.
    A job list page opens.
  3. Select a job or jobs for the analysis.
  4. Tap Next.
    The New Analysis page opens.
  5. Select analysis type and settings.
  6. Tap Done.
    The analysis is added to the list of analyses.

To view an analysis, follow these steps:

  1. From a Project page, tap on Analyses.
    The Analyses page opens.
  2. Tap an Analysis.
    The list of associated jobs opens.
  3. Tap a job to view the specific analysis.
