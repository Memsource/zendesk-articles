Introduction

Connectors provide a way to move translation texts between Memsource accounts, Content Management Systems (CMSs) and other online repositories.

Connectors are used to:

  • Manually import jobs directly into a project and export the translated files back.
    • Import jobs by going to Create New Job and selecting Add from Online Repository. 
    • Export jobs by selecting Download Files and clicking on Export to Online Repository.
  • Set up Automated Project Creation to automatically import new files from the connectors.
  • Set up an Automation Widget and let Submitters with special user rights upload files from connectors via the Client Portal.
