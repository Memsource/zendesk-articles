Note that as of this release the versioning strategy has changed. Version number will reflect the year of the release followed by the number of the release during the year. This release is the 22nd of the year 2019.

New Features and Improvements

Exclude specific keys from JSON import

During JSON file import it now possible to not only import specific keys, but also exclude specific keys (using regexp). 

Multilingual CSV

Multilingual CSV files can now be imported in a similar manner to Multilingual XLSX files.
