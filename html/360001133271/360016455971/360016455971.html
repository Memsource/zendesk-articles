New Features and Improvements

Improving Support for Bi-directional Texts in In-context Preview

Previously, right-to-left (RTL) texts remained aligned to the left despite the text direction being reversed. With this release, the support for bi-directional texts has been improved. Now, in the in-context preview, texts will be aligned according to the direction of the selected language.

Fixed Bugs

  • Using AltGr+Number did not work correctly for some national keyboards. This bug has now been fixed.
