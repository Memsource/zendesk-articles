v21.4.2 - March 2, 2021

Fixed Bugs

  • Tags are being incorrectly overwritten with next tag and text. This has been fixed.

  • Highlighting is incorrect when filtering by emoji. This has been fixed.

  • Emojis in the end of a segment cannot be replaced. This has been fixed.

  • Find Next and Replace are not working correctly when working with emojis. This has been fixed.

v21.2.3 - February 25, 2021

Fixed Bugs

  • This release includes general optimization improvements.

v21.2.2 - February 16, 2021

Fixed Bugs

  • Copying/pasting source text preceded by a tag resulted in the tag being pasted into the target segment along with the first character of the selection. This has been fixed.

  • Some segments in a job are being left empty when progressing from a previous workflow step. This has been fixed.

v21.0.2 - February 2, 2021

New Features and Improvements

  • Project Managers can now prevent Linguists from editing the content of tags.

Fixed Bugs

  • Tag metadata from TM is being ignored. This has been fixed.

v20.75.3 - January 19, 2021

Fixed Bugs

  • The cursor sticks on the Changes pane when confirming a segment. This has been fixed.

  • The editor freezes and requires a browser refresh to login again when adding to a TB. This has been fixed.

  • The word constructor cannot be used due to being a JavaScript keyword. This has been fixed.

  • Search results cannot be inserted into target by double-clicking or using keyboard shortcuts. This has been fixed.

  • The CTRL+arrow keyboard shortcut goes in the wrong direction in an RTL language. This has been fixed.

  • Incorrect quotation marks are used when typing in Dutch. This has been fixed.

  • Undoing results in joined jobs or locked segments results in incorrect display. This has been fixed.

  • Japanese characters are not being displayed correctly. This has been fixed.
