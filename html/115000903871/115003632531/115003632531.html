All users of Ultimate and Enterprise editions can utilize the Language Quality Assessment (LQA) feature for conversations in the Memsource Editor. Users are able to leave LQA evaluations for a segment or its parts to provide feedback, in a way similar to how the comments feature works.

Users can select one of the categories and subcategories that are set by the Administrator in the LQA Settings (for example Accuracy, Fluency, Terminology, etc.). Each of these categories can then be set to various levels of severity: Neutral, Minor, Major, or Critical.

Segments with added LQA are marked with "Q" in the Conversation column.
