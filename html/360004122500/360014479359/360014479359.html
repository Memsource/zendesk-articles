To access options for managing security environment and data access, follow these steps:

  1. From the Setup  page, scroll down to the Project Settings section and click on Access and Security.

    The Access and Security page opens.

  2. Select required options:

    • Email Notifications

    • Allow loading of external content in editors

    • Notify job owner when job status is changed

    • Memsource Editor for Web enabled for linguists

      Project Managers can control which environments their Linguists and Vendors may work. With this box checked, translators can click on file names and open them in Memsource Editor for Web.

    • Linguists may download their translation jobs

      Linguists and Vendors may download bilingual content for translation outside of Memsource. Bilingual MXLIFF files, bilingual DOCX files, and TMX files may be downloaded for translation in the Memsource Editor for Desktop or other external editor.

    • User metadata displayed to linguists in Memsource Editor

      Linguists and Vendors may see who translated identified TM matches. Deselect to provide anonymity.

    • Grant access to Memsource technical support

      Memsource Technical Support has access your account to help resolve technical issues. 

    • Activate/deactivate linguists automatically

      Some editions enable the creation of an unlimited number of users and have them activated and deactivated automatically based on task assignment.

    • Use vendors for projects

      Allow the use of Vendors that facilitate the sharing and collaboration of files across organizations.

    • Others may discover my organization as a Vendor

      Allows visibility on a Job Board.

    • Linguists may start a workflow step only if the preceding step has been completed

      Restrict or allow translators and reviewers to work simultaneously on one job or not until the job is set to Complete. See Workflow Overview.

    • Linguists may edit locked segments

      Linguists and Vendors are permitted to edit locked segments by clicking F2.

    • Linguists may edit source

      Linguists are permitted to edit source texts including the splitting and joining of segments.

      If not selected, Linguists are not permitted to:

      • Edit the source language in the Editor. Pressing F2, triggers the message Source edits disabled by project owner.

      • Upload bilingual MXLIFF or DOCX files. This includes any source modification in the Memsource UI, API, or Editor. Attempting to do so triggers the message Source edits prohibited by project owner. Upload failed.

      • Join or split segments.

    • Users may turn on/turn off auto-propagation of repetitions in Memsource Editor

      Permits Linguists and Vendors the option of enabling or disabling the Auto-propagation of Repetitions feature in Memsource Editor.

    • Users may turn on/turn off Instant QA in Memsource Editor

  3. Set Session timeout value.

  4. Set Trusted IP Addresses (Ultimate and Enterprise editions only)

    If specified, users are not able to access their accounts from other addresses. Select Advanced to set access options based on user roles.

  5. Click Save.

    Preferences are applied.
