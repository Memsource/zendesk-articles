Bitbucket Cloud is similar to Bitbucket Server, but is different in terms of authentication and access. Below, you’ll find instructions on how to integrate your Bitbucket Cloud account and your Memsource account.

 

Note: You will need to have pop-ups enabled in your browser in order to successfully connect your accounts.

Configure Bitbucket Server

Before setting up the connector in Memsource, you’ll want to log in to your Bitbucket account in order to create a personal access token. This will be used later in Memsource. To create this token, click on your profile picture and select Manage account from the dropdown list.

Under Account settings, select Personal access tokens and then click the Create a token button. Enter a token name (this can be anything). Then, set the permissions as follows: Project—Read, Repositories—Write. Then, click Create.

Important: When the pop-up window with the password appears, copy and save it somewhere, as it will not be displayed again.

 

The last thing you’ll need is your Bitbucket Server account’s URL. This will be everything in the address bar up to the first slash (/). Copy this and save it alongside your personal access token.

Set up the Connector in Memsource

In Memsource, navigate to Setup and scroll down to the Integrations section. Click on Connectors. After clicking on New, you can choose the connector you wish to create. Give it a name and change the Type to Bitbucket Server.

Enter your Bitbucket site URL and your personal access token into the fields provided and click Test Connection. If you've entered everything correctly, a little check mark will appear. Then, click Save.
