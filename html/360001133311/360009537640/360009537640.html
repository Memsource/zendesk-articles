New Features and Improvements

In-context Preview for IDML Files (Beta)

The In-context Preview feature in the Memsource Editor is now also availble for INDD and IMDL files.

 

Concept-level Metadata for Term Bases

In the Edit Concept section under Edit Term, concept-level metadata can be added for terms in term bases. It is possible to specify the Domain, Subdomain, URL, Definition, and Concept Note. Such concept-level metadata is now also displayed within the Memsource Editor in the CAT Pane metadata section.

 

Metadata Section Improvements

The metadata sections for translation memories, term bases, non-translatables, and machine translation matches within the Memsource Editor have been redesigned to make the information more visible.

 

Fixed Bugs

  • Due to a bug, scores would be shown for NT segments, even if this feature was turned off. This has now been fixed.
  • When confirming a segment after using a filter (e.g. "not locked" or "not confirmed), the cursor would not always jump to the next unconfirmed segment. This has been resolved.
