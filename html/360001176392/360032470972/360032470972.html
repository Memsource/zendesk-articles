New Features and Improvements

Updates Made to the "Unedited Fuzzy Match" Error

For segments containing numbers only, the Unedited Fuzzy Match error will no longer be displayed during the QA check.

 

Contentful (Entry-level) Connector Now Available

Since there are multiple ways to localize content using Contentful, we now have a connector available to use for entry-level content in Contentful. Previously, there was only a connector available for field-level content.

 

Fixed Bugs

  • Due to a bug, relevancy data would sometimes not be displayed on the Edit page for Linguist users. This has now been resolved.
  • When creating a project from a template, some of the parameters would occasionally be missing. This has been fixed.
  • When adding reference files to a project created with the Automation Widget, an email would sometimes be sent to the Submitter saying that the files were unsupported. This email is no longer sent out when the files are supported.
