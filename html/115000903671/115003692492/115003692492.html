Quotes are automatically generated for jobs created with the Automation Widget in all editions (except the Personal edition). For Ultimate and Enterprise editions, they can be also created manually from Analysis.

In order to create a New Quote, you have to have the Analysis first.

  • In the project, select New in the Quotes panel.
  • Set a Name (optional, if empty, the default name "Quote #1" etc, will be used)
  • Select Analysis.
  • Select a Provider (Linguist or Vendor). The quote will be visible to the Provider in their Memsource account. The Provider will be listed in the Quote Report results. The field can be empty, if you don't want to assing it to anyone.
  • Select Net rate scheme (optional).
  • Select Price list (optional, you can add the price in the following step).

In the next window:

  • Select Currency (this can be pre-selected from the Price list in the previous step).
  • Select Unit (this can be pre-selected from Price list in the previous step).
  • Enter a Price (this can be pre-selected from the Price list in the previous step).

In the last window, you can check the Quote and Save it.
