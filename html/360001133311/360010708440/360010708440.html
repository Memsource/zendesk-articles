Fixed Bugs

  • Loading of in-context preview was extremely slow for large jobs. The preview has been optimized to speed up the loading.
  • Some segments were incorrectly flagged as having Instant QA issues after batch QA was run. This has been fixed.
  • Copying text from any place in the Editor nor pasting text to Editor did not work in Microsoft Edge Windows 10. This has been fixed.
  • In large jobs, the Go to segment feature would not function when selecting an unloaded segment. This has been fixed.
