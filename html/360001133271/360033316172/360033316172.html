New Features and Improvements

Arial Font is Used for Arabic

Arial font is now the primary font used for Arabic in the editor.

 

Fixed Bugs

  • Due to a bug, it was not possible to run the QA check if the dropdown menu was open. This has now been fixed.
  • When searching a TM or TB using a concordance search, if a new search was done that returned 0 results, the results from the first search would not disappear. If the subsequent search has 0 results, the Search Pane will now be empty.
  • Due to a bug, it was not possible to use the scroll bars when using the touch screen on a Windows tablet. This has now been resolved.
  • MT and TB terms that had subscript letters or numbers (such as the 2 in H2O) would not be found when using a search. This has now been fixed.
