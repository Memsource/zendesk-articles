A Project Template is used in project creation. Repeating project criteria is maintained for regular clients, projects shared with Vendors, or projects created with the Automation Widget. Use of a template streamlines the submission process and reduces errors.

Create a Project Template

Project Templates are created from existing projects.

To save a current project as a template, follow these steps:

  1. Open an existing project.

  2. Click Save as and select Save as a Template.

    The Create Project Template page opens.

  3. Provide a name and change project settings if required.

  4. Click Create. Template is added the list of Project Templates.

Manage Project Templates

To edit a project template, follow these steps:

  1. From the Setup  page, scroll down to the Automation section and click on Project Templates.

    The Project Templates page opens.

  2. Click on a project.

    The Edit Project Template page opens.

  3. Change project settings and click Save.

    The changes are applied to the selected project.

The list tools in the upper right corner of the list are used to sort, filter and customize the list of Project Templates.

Using Project Templates

To create a new project using a template, follow these steps:

  1. From the Projects page, click New.

    The Create Project page opens.

  2. From the Use Template field, select a template from the dropdown list.

    The project is populated with options specified in the template.

  3. Change options if required.

  4. Click Create.

    A new project is created with specified options.

Assign Providers from a Template

Project Managers can assign providers (Linguists or Vendors) to each of the project workflow steps using an existing Project Template.

The number of workflow steps and the source language in the chosen project template must match those in the new project or the project template will not appear in the list. Target languages can be different, but at least one of the target languages must be the same. 

Assigning providers from a template will not overwrite any existing assignments. Providers can be assigned from multiple templates as long as the templates match the criteria.

To assign a provider from a template, follow these steps:

  1. From the Job Table, either:

    • Select specific jobs to assign providers

    • Select none to either assign providers to all jobs

    • Select none to assign some providers to some jobs across all workflows.

  2. Click Tools and select Assign providers from a Template.

    The Assign providers from a Template window opens.

  3. Select a project template from the dropdown menu and optionally to Apply providers to just the selected jobs.

    A list of defined workflow steps and potential providers is displayed. Clicking on the workflow step presents provider details.

  4. Click Assign.

    Providers from selected template are assigned to the selected job(s).
