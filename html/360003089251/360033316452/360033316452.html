New Features and Improvements

Landscape View is Now Available

It is now possible to edit segments in landscape view, making translating easier and more convenient.

 

The "Cancel" Button Has Been Replaced

In order to streamline the look of our Mobile Editor, the Cancel button in the top right corner of the screen where segments are edited has been replaced by an X on iOS devices. This button will still function in the same way as the Cancel button had.

 

Terms Found in the Term Base are Now Highlighted

In keeping with the Editor for Web and the Editor for Desktop, terms that are found in the TB will now be highlighted when looking at a segment. Note that this is only available for Android devices.
