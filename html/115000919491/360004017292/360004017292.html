If you’re a Project Manager on the move and you want to check the status of the projects you are currently working on, everything you need to keep on top of your organization’s projects is available through the Project Dashboard in the Memsource Mobile App.

 

Review

The dashboard categories provide a quick summary of the project statuses. From here, you can view each category and the details of the projects within them. This is particularly useful if you want to view the statuses and due dates of jobs within your projects. You can also view the resources attached to the project.

Edit

From the Project Details page, you can add new jobs and select a translation memory or term base for the project by pressing the plus (+) button. (Note: You can only select existing TMs or TBs.) See the Creating New Projects and Adding Jobs article for more information.

The key project settings can all be viewed and edited by pressing the three dots (...) in the top right-hand corner and selecting Edit.

In addition to these, you can also edit the jobs inside of your projects. Please see our Editor for Mobile article for more information.
