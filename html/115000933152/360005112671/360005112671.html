If you are new to Memsource and are a Project Manager or Admin user, this Getting Started Guide is for you. We also have a dedicated Getting Started Guide for Linguist users.

This Getting Started Guide for Project Managers will take you through everything you need to know to get up and running in Memsource. It includes step-by-step instructions and concise demo videos so you will find your way around in no time.

Before you get going, just a quick note about the structure of Memsource.

Our tool comes in three parts:

  1. Memsource UI: A web-based environment where Project Managers can create and maintain projects.
  2. Memsource Mobile App: From the app, you can manage your translation projects on the go. Download it for free for Android and iOS devices.  
  3. Memsource Editor: This is where the actual translations are done. The desktop editor app is available free for download and the web editor can be accessed from Memsource.

This guide will focus on the Memsource UI. The Getting Started Guide for Linguists provides an overview of Memsource Editor.
