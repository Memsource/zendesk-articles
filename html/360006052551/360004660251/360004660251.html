Introduction

DocBook is a semantic markup language for technical documentation. It is a standard format that is defined and maintained by the DocBook Technical Committee of OASIS.

The DocBook file filter by default imports files with the following extensions:

  • .dbk

Default Import Settings

DocBook is imported with default settings according to DocBook format standards.

Custom Import Settings

Default DocBook settings will be imported. Users can specify additional elements separated by a comma (title, body, etc.) in the following fields:

  • Add translatable elements - Default translatable elements will be imported.
  • Add non-translatable elements - Default non-translatable elements will be skipped.
  • Add translatable inline elements - Default translatable inline elements will be imported.
  • Add non-translatable inline elements - Default non-translatable inline elements will be imported.

Tag Content Visualized

The tag content of the source .dbk file can be visualized by clicking on Edit and selecting Expand Tags in the Editor. They can be edited by pressing F2.
