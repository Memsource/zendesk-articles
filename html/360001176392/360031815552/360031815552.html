New Features and Improvements

Include Tags Checkbox for Amazon MT

There is now an Include Tags checkbox available for Amazon Machine Translation.

 

Set a Job to Completed Once All Segments are Confirmed

There is now an option to set pre-translated jobs to Completed once all segments in that job are confirmed. This option is only available for jobs where Set segment status to confirmed is selected.

 

Using Analyses for Projects with Vendors

When using a Vendor in a project, it is now possible to sort your analyses by each Vendor/Linguist. The by Linguist parameter has now been changed to by Provider to reflect this.

  

Pricing Information for Users and Vendors

Pricing information and Net Rate Schemes can now be shown for both Users and Vendors in our REST APIs.

 

Fixed Bugs

  • Due to a bug, some tokens and HTML code would not be converted into tags when using Marketo. This has now been fixed.
  • When translating long segments using the DeepL MT, the translation would be cut off. This has been resolved.
  • For some users, the page to edit users would load extremely slowly. This page should now load at a normal speed.
  • Some users were not able to submit files using the Automation Widgets on some sites. This has now been resolved.
