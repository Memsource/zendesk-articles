The CSV file filter by default imports files with the .csv extension.

File import settings that can be selected when creating a new job:

  • Use HTML subfilter
  • Column delimiter
  • Import specific columns only
  • Context key - see TM Match Context and Optimization for more details.
  • Context note (column numbers separated by comma) - Memsource can display more comments from different columns. Each comment will be on a new line (in the Editor's Context window)
  • Max target length
  • Convert to Memsource tags (use regexp): Matching text will be converted to Memsource tags
