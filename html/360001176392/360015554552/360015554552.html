New Features and Improvements

New Webhook for Due Date Changes

Until now, only three webhook events were supported in Memsource (Job created, Job assigned, and Job status changed). This release is adding a new event (Due Date changed) which allows users to get notified every time a job due date has been changed in a project.

 

New Languages Supported in Amazon Translate

The list of supported language combinations for Amazon Translate has been extended to include the following languages (in combination with English):

  • Chinese (Traditional)
  • Czech
  • Italian
  • Japanese
  • Russian
  • Turkish

Fixed Bugs

  • It was not possible to select the desired MT engine using Internet Explorer. This bug has now been fixed.
  • Due to a bug, discrepancies sometimes occurred when comparing the analysis results in the UI and downloaded LOG files. This has now been resolved.
  • In some cases, it was not possible to delete translations from a translation memory using the Memsource UI. This bug has now been fixed.
