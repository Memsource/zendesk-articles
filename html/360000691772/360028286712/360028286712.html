Configure the Connector in Help Scout

In order to establish a connection between your Help Scout account and Memsource, a Help Scout Docs API Key is required. To get this, log into your Help Scout account. Select Your Profile from the Account dropdown list in the top right-hand corner of the dashboard.

 

Select Authentication on the right side of the screen and then select the API Keys tab. Under the Docs API section, click Generate an API Key. Copy this key and save it somewhere.

Set up the Connector in Memsource

In Memsource, navigate to Setup and scroll down to the Integrations section. Click on Connectors. After clicking on New, you can choose the connector you wish to create. Give it a name, change the Type to Help Scout, and paste your API key into the Help Scout Docs API key field.

In the Help Scout Target Collection field, type the name of an existing Help Scout collection, or the name of the new collection you want to create. If you create a new collection, this new collection will also appear in the list of collections in your Help Scout account.

 

Note: To view a Help Scout collection created via the connector in Memsource, the collection must first be saved. Do this by navigating to Help Scout, clicking on Manage, and selecting Docs. Then, select the target site for the collection. Note that in a Basic or Standard account, there will only be one site. In a Plus account, there will be multiple sites. Select Collections on the left-hand side and click on the collection created by the connector. Finally, click Save Collection.

Once these fields are completed, click Test connection.

For a Basic or Standard Help Scout account, a message will appear to confirm the connection to the single site on Help Scout.

For a Plus Help Scout account, a list of your multilingual sites will be displayed. Select the language of each site from the drop-down lists. You will also need to select which site will be the source site for the translation.

Translate Articles from Help Scout

Please note that when an article is first translated and exported back to Help Scout, the article will appear as a draft and will need to be published manually. However, if an article already published in Help Scout is translated in Memsource, once it is exported back into Help Scout the article will immediately be published.
