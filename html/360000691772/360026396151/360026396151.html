Joomla is a Content Management System (CMS) that allows users to create and maintain web content. The Joomla Connector allows you to connect to the Joomla CMS with Memsource. Using Memsource, you will be able to translate your articles and categories by either manually uploading them or by using  Automated Project Creation to initiate jobs based on rules that you define in a Project Template.  This article provides a step-by-step guide to setting up and using the Memsource extension for Joomla.

Requirements

Before you begin, you'll need to install a plugin into your Joomla instance. This plugin uses Joomla Multilingual Associations, which are supported in Joomla 3.7 and newer. 

Set up the Connector in Joomla

In order to install the plugin, log into your Joomla account. Find the Memsource Connector in the Joomla Extensions Directory by clicking on Components and selecting Memsource Connector. On this page, you'll see your Memsource Connector Token. Copy this to your clipboard.

You should also copy the URL from your browser. Everything up to and including the ".com" will be used as the Host name in Memsource.

Set up the Connector in Memsource

Navigate to Memsource and log in. Go to the main Setup and scroll down to the Integrations section. Click on Connectors. After clicking on New, you can choose the connector you wish to create. Give it a name and change the Type to Joomla. 

Enter the URL you copied from Joomla into the Host section. Paste the token from your Joomla account into the Token field. Then, click Test Connection.

If you’ve done everything correctly, you should see a little checkmark next to the Test Connection button. After this, click Save.
