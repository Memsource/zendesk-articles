New Features and Improvements

Support for In-context Preview for Multilingual XML files

Like monolingual XML files, the multilingual XML format now also supports previews. Attach the appropriate XSLT template, which can generate an HTML preview (via the Preview translation option in the Editor) instead of the XML one. You can also use it as an in-context preview in the Editors.

You can find out more information in our multilingual XML documentation.

 

New Sitecore Connector

The Sitecore connector is the latest addition to our range of integrations and is available for our Enterprise users.

Sitecore is a customer experience management software that includes a content management system. The Memsource connector for Sitecore pulls web content from Sitecore folders into Memsource for translation. The translated files are then pushed back into Sitecore. This connector can also be used as part of the Automated Project Creation.

To learn more about how to set the connector up, please visit our Connectors documentation for Sitecore.

 

Marking Terms as Preferred

When working with complicated terminology, several translations of a term can be available to the translator. It is now possible to mark terms as Preferred, indicating which of the available terms should be used in the translation.

Preferred terms will always be displayed on top of the list of terms in the CAT panel, or they can be identified by an additional remark in the suggestion details.

See our Term Base article to learn more about term attributes like Preferred.
