New Features and Improvements

Importing Columns from Microsoft Excel

It is now possible for users to select which columns they would like to import from Excel files without having to import the entire file. This can be done by using a list of columns separated by commas (e.g. A, B, F, G). These columns will be imported from every sheet in the document.

 

Assigning Automation Widgets Via API2

Users can now assign Automation Widgets via API by using the following: PUT api2/v2/users{userId}.
