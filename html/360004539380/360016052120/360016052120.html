v20.22 - November 17, 2020

Fixed Bugs

  • REST API:

    ZIP files cannot be imported to a TM. This has been fixed.

    Running an analysis of an empty job causes a failure. This has been fixed.

    Exact search not working correctly for exportByQueryAsync. This has been fixed.

  • Google Drive: Not all folder types are visible. This has been fixed.

v20.21 - November 3, 2020

New Features and Improvements

  • Marketo: Modules can now be excluded from translations.

  • WordPress Plugin 2.11: Now compatible with WordPress 5.5 and PHP 7.4

Fixed Bugs

  • AEM 3.0.8: An error occurred when saving language mapping. This has been fixed.

  • GitLab: Cancelled jobs are being exported back to the repository. This has been fixed.

  • GitHub: APC jobs are not being pushed to the repository. This has been fixed.

  • WordPress Plugin 2.11: Issues with exporting jobs back to WordPress and Scan now and adjust links reverting translation have been fixed.

v20.20 - October 20, 2020

New Features and Improvements

  • API: GET /api2/v1/projects/{projectUid}/jobs/{jobUID} and GET /api2/v2/projects/{projectUid}/jobs endpoints return the path of the file in the archive in the originalFileDirectory parameter.

Fixed Bugs

  • FTP/SFTP, Contentstack, Git, Kentico Xperience, and Sitecore connectors now securely handles passwords when editing configurations.

  • Zendesk Connector: Not all projects are being created when using Automated Project Creation. This has been fixed.

v20.19 - October 6, 2020

New Features and Improvements

  • Sitecore Connector: url and other non-translatable attributes in General Links and Image stay unchanged in Sitecore after updating an existing translation.

  • Git Connector: When Automated Project Creation is set to monitor subfolders, the target folder maintains the same subfolder structure as the source folder.

Fixed Bugs

  • API: pageNumber param not working correctly. This has been fixed.

  • API: Guests can change job's status via API when not permitted via the UI. This has been fixed.

  • Google Drive Connector: Not all subfolder levels are available. This has been fixed.

v20.18 - September 22, 2020

New Features and Improvements

  • The API endpoint for Update source is now available.

Fixed Bugs

  • Git Connector: Continuous Job check fails if a project contains multiple target languages and the target file is locked and can't be updated. This has been fixed.

  • Google Drive: Subfolders in shared drives are not listed when selecting a remote folder from the connector or creating a new job. This has been fixed.

v20.17 - September 8, 2020

New Features and Improvements

  • AEM 2.1.0: Memsource project number and UID used in REST API are stored with the AEM job.

  • Marketo: When Translate tokens setting is on in the connector configuration, Marketo tokens are pulled in for localization, otherwise they are excluded.

  • Google Drive: Subfolders in shared drives are not listed. This has been fixed.

  • API: A new PUT endpoint for editing existing Pre-translate and Analyse settings in project templates has been added.

Fixed Bugs

  • AEM 2.0.5: Plugin was not compatible with AEM default minifier. This has been fixed with updated plugin (2.0.8).

v20.16 - August 25, 2020

New Features and Improvements

  • API:

    • POST /api2/v1/projectTemplates endpoint now creates new independent instance of import settings for the project template.

    • New GET /api2/v1/projectTemplates/{projectTemplateId}/importSetting endpoint returns import settings associated with a project template.

Fixed Bugs

  • Patch job would not assign provider if status not specified. This has been fixed.

v20.15 - August 11, 2020

New Features and Improvements

  • Webhook authentication is now supported.

  • AEM (2.0.6): Support for {order.title} macro in the Automation Widget. Title field to represent Adobe Experience Manager project title and {order.email} to represent email of the Adobe Experience Manager submitter. If the email is not set, then email used in Memsource credentials specified in the Adobe Experience Manager Cloud Services is used.

Fixed Bugs

  • Contentstack connector: Fields marked as non-localizable fields are still being localized. This has been fixed.
