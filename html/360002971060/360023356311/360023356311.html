Memsource Translate is currently in beta. Try it now. 

Memsource Translate (Beta) evaluates the quality of different machine translation engines and automatically selects the best performing engine for a project, based on the language pair. It supports 69 language pairs and includes Machine Translation Quality Estimation (MTQE) at no additional cost. MTQE is an AI-powered feature which assigns quality scores to machine translation output before any post-editing is done.

How it Works

Memsource Translate is available automatically in every account, so there’s no need to worry about activating it. If you have just signed up to Memsource, it will be selected as the default machine translation engine.

Every account will start with free characters to use.

  • Personal edition: 50k characters
  • Team Start edition: 250k characters
  • Team and Academic editions: 500k characters
  • Ultimate, Enterprise, and Enterprise starter editions: 2 million characters

Instead of Memsource Translate characters being tied to a specific MT engine, they can be used for all of the MT engines supported by Memsource Translate. (Currently, Microsoft Translator and Google Translate are available, with more to follow.)

Note: Currently, Memsource Translate does not support custom MT engines, and it is not possible to create multiple instances of Memsource Translate.

Activating and Deactivating Memsource Translate

Memsource Translate is active by default. If you decide to deactivate it, go to Settings, scroll down to the Integrations section, and click on Machine Translation Engines. In the Memsource Translate section, click on Configure. Here, you’ll find General Settings and Machine Translation Engines. Under Machine Translation Engines, you can check or uncheck the engines that you want to use, and then click Save.

Purchasing Memsource Translate Characters

As mentioned previously, when you enable Memsource Translate, you will receive free Memsource Translate characters that can be used for machine translation. After these are used up, you can purchase additional character bundles through Memsource Translate. Do this by going to your account’s Settings and clicking on Machine Translation Engines in the Integrations section. In the Memsource Translate section, you will see a button that gives you the option to Top Up Characters.  For a breakdown of our pricing for these character bundles, please see the chart below.

Remaining Characters

For Memsource Translate, the Remaining Characters column shows the number of characters available to use with Memsource Translate. As you use the characters, this number will decrease. Find out how to purchase more Memsource Translate characters.

Please note: If you don't have any remaining characters for Memsource Translate, you will not be able to use the Machine Translation Quality Estimation.
