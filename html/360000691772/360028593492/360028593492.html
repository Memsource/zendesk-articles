Memsource provides a Secure File Transfer Protocol (SFTP) connector. Like FTP, SFTP allows users to transfer files between different computers and acts in a way that is similar to how transferring files on your own computer work.  Using the SFTP connector, users will be able to translate these files either manually or through automated project creation. Below, you'll find instructions for how to set up this connector.

Setting up the Connector

To set up your connector, log into Memsource and navigate to Setup. Scroll down to the Integrations section. Click on Connectors. After clicking on New, you can choose the connector you wish to create. Give it a name and change the Type to SFTP.

Once you change the Type of connector to SFTP, you'll need to enter the following information:

  • Your Host name. This is your domain name and will begin with "ftp".
  • The Port number. This is automatically set by Memsource to 22 and can be left as it is.
  • Your SFTP User name. 
  • Your SFTP Password.
  • Your SSH key.
  • Your SSH passphrase.

Once you've entered the information above, click on Test Connection to see if the credentials are correct and if Memsource can reach the site successfully. If you've entered everything correctly, you should see a little check mark next to the Test connection button. Then, click Save.
