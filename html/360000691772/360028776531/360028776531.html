Marketo is a software company that helps users with their account-based marketing needs. With our Marketo connector, you will be able to translate emails, email templates, files, forms, landing pages, and landing page templates.

Notes Regarding Translations

For email templates and landing page templates where the name template name [locale] already exists, the template content will be modified. If it does not exist, the original template will be cloned and the template content will be modified. In both cases, the template will be approved.

For emails and landing pages, if an asset with the name asset name [locale] already exists, the content will be modified. If it does not exist, all data will be copied from the original asset and the content will be modified. A translated template will only be used if a translated template already exists and is approved. Otherwise, the original template will be used.

Set up the Connector in Marketo

In order to enable the communication between Memsource and Marketo, REST APIs need to be enabled in your Marketo account. To do this, please refer to the Marketo documentation.

Once the REST API is enabled, you'll need to get your Client ID and Client Secret. Do this by going to Admin and clicking on LaunchPoint in the Integration section. Click View Details. Copy and save your Client ID and Client Secret.

Close the dialog box and navigate back to the Integration section of the Admin page and select Web Services. In the REST API section, find the Identity row. It will have a value like http://000-ABC-111.mktorest.com/identity. Copy and save the Host Prefix (in this case 000-ABC-111).

Set up the Connector in Memsource

Navigate to Setup and scroll down to the Integrations section. Click on Connectors. After clicking on New, you can choose the connector you wish to create. Give it a name and change the Type to Marketo.

In the Marketo API key field, enter your Marketo Client ID. In the Marketo API secret field, enter your Marketo Client Secret. Finally, enter the first part of your Marketo API URL into the Marketo API identity URL field.

After you've entered all your information, click the Test connection button. If you've entered everything correctly, you should see a small check mark next to the Test connection button. Then, click Save.
