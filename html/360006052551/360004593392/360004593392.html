The Adobe Photoshop file filter, by default, imports files with the .psd extension. The following import settings can be adjusted for Photoshop files:

  • Import hidden layers
  • Import locked layers

Note: Because Photoshop does not automatically update documents with their new translations, you must do one of the following:

  • When opening the PSD file in Adobe Photoshop, confirm the dialog box asking for updates to certain layers. 
  • If the dialog box does not show up or does not update all the translated content, click the layers containing the translated text in order to update them.
