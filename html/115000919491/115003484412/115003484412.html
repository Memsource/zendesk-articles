Memsource Mobile App allows users to directly connect to their Memsource account on their mobile phones and similar devices. 

The App currently allows users to both create and manage their ongoing projects on the go, work with existing Translation Memories and Term Bases and/or view Analyses. 

 

 

To learn more about the Memsource Mobile App, you can read our dedicated blogpost - Announcing the Memsource Mobile App - or visit the section or the dedicated section of our documentation - Memsource Mobile App.

Here are basic instructions on how to work with the Memsource Mobile App. 

 

Project Managers

Project Managers using the Mobile App can see the overview of the projects created in the linked Memsource account, and manage the projects directly from the application.


Navigation

The top panel shows All, Overdue, In Progress and My projects. This gives Project Managers easy access to all the relevant projects at hand. Due dates are displayed on the right hand side of the screen, with overdue project dates highlighted in red. Approaching deadlines are also indicated in yellow.

Project Creation

Using the + button it is possible to create new projects directly from the App. This allows Project Managers to create projects using their devices. A simplified UI also includes the selection of existing metadata (Clients, Domains, etc.), using Project Templates or setting the project Due Date. 

Project Management

Opening a project will bring up the Project interface (containing information like the source and target languages, selected workflow steps, due date, etc.), plus dedicated sections for Translation Memories, Term Bases, Jobs and Analyses. 

Jobs, TM and TB sections allow users to browse the attached resources, and to attach additional ones (TMs and TBs can only be selected from those already created in the account, Jobs can also be imported directly). It is possible to add files directly from the mobile device (Memsource Mobile App will be suggested as an option for the supported file formats).
Analyses need to be created in the accounts first before they can be viewed in the Mobile App.
 

Linguists

Navigation

Similar to Memsource Cloud UI, Linguists see a different navigation panel in the Mobile App. The panels available in the App are the same as in the Memsource Cloud UI - New, Accepted, Completed projects. 

Managing Assignments

Once Linguists receive an email notification, they no longer need to go to the Memsource Cloud UI via their browsers. They can use the App to directly view assignments at hand, check important details like language combination and due dates and directly accept new assignments in the application itself. 

The Project view shows project Due Dates as well as color warnings for approaching deadlines.
