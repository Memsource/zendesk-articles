Memsource Translate (Beta) evaluates the quality of different machine translation engines and automatically selects the best performing engine for a project based on the language pair. It supports 69 language pairs and includes Machine Translation Quality Estimation (MTQE). 

Memsource Translate is provided in all accounts and is the default machine translation engine.

  • Memsource Translate characters can be used by all MT engines supported by Memsource Translate.
  • Machine Translation Quality Estimation (MTQE) requires Memsource Translate characters.
  • Memsource Translate does not support custom MT engines.
  • Multiple instances of Memsource Translate can not be created.

Every account starts with free characters:

  • Personal edition: 50k characters
  • Team Start edition: 250k characters
  • Team and Academic editions: 500k characters
  • Ultimate, Enterprise, and Enterprise starter editions: 2 million characters

Activating and Deactivating Memsource Translate

Memsource Translate is active by default.
To deactivate Memsource Translate, follow these steps:

  1. From the Setup page, scroll down to the Integrations section. Click on Machine Translation Engines.
    The Machine Translation Engine page opens
  2. In the Memsource Translate section, click on Configure.
    General Settings and Machine Translation Engines are presented..
  3. Under General Settings, deselect Make default.
  4. Click Save.

Purchasing Memsource Translate Characters

When Memsource Translate is enabled, the free Memsource Translate characters that can be used for machine translation are made available. When depleted, additional character bundles must be purchased.
To purchase character bundles, follow these steps:

  1. From the Setup page, scroll down to the Integrations section. Click on Machine Translation Engines.
    The Machine Translation Engine page opens
  2. Click Top Up Characters.  
    The character purchase page opens.
  3. Select currency, payment method, bundle and location options.
  4. Click Continue.
    The billing data page opens.
  5. Provide billing data and click Continue.
    The review page opens.
  6. Ensure billing data is correct and click Continue.
    Payment gateway opens.
  7. Complete payment process to finish purchase of characters.
    Purchased characters will be added to the Remaining Characters column showing the number of characters available for use with Memsource Translate.
