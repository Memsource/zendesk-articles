New Features and Improvements

Choose Characters that Should Not Be Escaped in MD Files

When working with MD files, users can now decide which of the characters should not be escaped when the translated MD file is exported from Memsource. This can be especially helpful with more complicated MD structures or formattings.

 

New MT Engine—Mirai Translator

Mirai Translator is now supported as an MT engine in Memsource.

 

Remove All Non-Memsource Tags Renamed

When importing a Translation Memory, the option Remove all non-Memsource tags has been renamed to the more general title Remove tags, but the functionality has remained intact.

 

Quote Report Improvements

A new section has been added to the Quote Report page, allowing users to add the following information into the generated CSV file:

  • Business Unit
  • Client
  • Cost Center
  • Domain
  • Subdomain
  • PO number

 

Adjustments to Wordcount Charts on Home Page

The Wordcount charts will now display the number of source words processed through Memsource.

Fixed Bugs

  • If a user accepted a split job (where other users have been assigned as well), others had been incorrectly removed from their respective jobs. This bug is now fixed.
  • Projects, TMs, and TBs whose name contained an underscore (_) had not been correctly filtered by it. This has now been resolved.
  • The rate in the Job Report was incorrectly rounded to 2 decimal points. This has now been fixed.
  • Due to a bug, the target file could not be exported into OneDrive repositories. This has now been fixed.
