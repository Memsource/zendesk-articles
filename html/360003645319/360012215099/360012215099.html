Services are used in Automation Widgets and in Automated Project Creation. They define a service product made available to a submitter with set parameters such as languages available, billing units, rates, etc.. They function as templates applied to automatically created projects.

Set up Services

To set up services, follow these steps:

  1. From the Setup  page, scroll down to the Automation section.

  2. Click Services.

    The Services page opens.

  3. Click New.

    The Create Service page opens.

  4. Complete required fields:

    • A public and private name

      The public name is displayed in the widget with the private being the internal name.

    • Type of service

      Professional Translation is the only option.

    • Source and Target languages

    • Project Templates

      Select appropriate Project Templates to be used when this Service is selected. Only add templates with languages fitting the Service languages.

      If multiple templates are selected, the system will select the one that best fits the selection from the Service; an EN->DE template will not be used for EN->FR translations.

      Workflow steps in the template must also be set in the service.

    • Buyer

      If this Service will be used by a specific Buyer.

    • Financials

      Analysis and quote creation options.

    • Billing Unit

    • Net Rate Scheme

    • Due Date Scheme

      Due Date will be automatically calculated (The Date Picker will be disabled in public Automation Widget page.)

    • Job Due Date

    • Price List

    • Workflow Steps Not available in all editions.

      Memsource projects created via this Service will include the specified workflow steps.

      The workflow steps in the Service must contain same or more steps as the Project Template.

  5. Click Save.

    The new Service is added to list.
