Under Setup, click on Access and Security. Here, users will find several options that are designed to create a better-controlled environment and to help manage the security of your data.

Web Editor Enabled for Linguists

This option helps Project Managers to control in which environment their Linguists and Ventors will be able to work. With this box checked, translators will be able to click on file names and open them in Memsource Editor for the web.

Linguists May Download their Translation Jobs

With this box checked, Linguists and Vendors will be able to download bilingual content for translation outside of Memsource. That is, translators will be able to download bilingual MXLIFF files, bilingual DOCX files, and TMX files for translation in the desktop editor.

User Metadata Displayed to Linguists in Memsource Editor

If you would like your Linguists and Vendors to be able to see who translated the TM matches they are getting, check this box. If you would rather have this data be anonymous, make sure this box is unchecked.

Grant Access to Memsource Technical Support

By default, Memsource Technical Support can access your account if you need help resolving a problem. If this would make you feel less secure, untick this option.

Activate/deactivate Linguists and Vendors Automatically

Some editions enable you to create an unlimited number of users and have them activated and deactivated automatically based on task assignment.
If you wish to make use of such automatic activation and deactivations, leave this option ticked.

Use Vendors for Projects / Others May Discover my Organization as a Vendor

Some editions of Memsource allow the use of Vendors that facilitate the sharing and collaboration of files across organizations. These two options allow you to make use of external translation providers or to work only with internal users.

 

Note: Memsource has a Job Board that makes it possible for users to discover each other as potential partners. Checking the Others May Discover my Organization as a Vendor option will make sure that you are visible to others if you wish. If you do not wish to be visible to others, make sure this box is unticked.

Linguists may Start a Workflow Step Only if the Preceding Step has Been Completed

With this option, you control whether you want your translators and reviewers to work one after another, or if they will be able to work simultaneously. You will find more details about workflow configuration and the possibility to run workflow steps simultaneously in our Workflow article.

Linguists may Edit Locked Segments

With this option disabled, Linguists and Vendors will not be able to use the F2 key to edit locked segments.

Linguists may Edit Source

This option controls whether or not linguists can edit the source text, including splitting and joining segments.

Linguists may Turn On/Turn Off Auto-propagation of Repetitions

Using this option, Admin and Project Manager users can choose whether Linguists and Vendors have the option to enable or disable the Auto-propagation of Repetitions feature in Memsource Editor.

Session Timeout

All users of an organization will be logged out of Memsource after a certain amount of time if they are idle. The time can be customized by Admin and PM users and can range from 15 minutes to 72 hours.

Trusted IP Addresses (Available in Some Editions Only)

Ultimate and Enterprise editions of Memsource allow you to ensure that your users can only their accounts from certain IP addresses. This is the section where you can define these trusted IP ranges. With this enabled, users will not be able to access their accounts from other locations, even if they have a valid username and password.

There is also "option if you want to define trusted IP addresses for your admin and project management team only, while leaving linguist users unaffected.

Two-factor Authentication

There is another important security-related feature which you will find under Setup by clicking on User Settings: Two-factor Authentication.

The Two-factor Authentication, also called Two-step Authentication, can considerably increase the security of your user account. This will be linked with Google Authenticator, Authy or another compatible application on your phone.

Once it is set up, your account will be linked to the app on your phone, and when you sign in to your Memsource account, you will be prompted to provide your username, password and a code that you receive on your phone.

To set up Two-factor Authentication:

  1. Download Google Authenticator or a similar app to your smartphone.
  2. In Memsource, go to Setup, click on User Settings and click Two-factor Authentication.
  3. Scan the QR code with your phone, and the app will display a series of numbers (e.g. 123 456).
  4. Enter the code from your phone into the Code field.
  5. Click on Verify. In order to do this, your phone must be connected to the internet.

If the QR code is not displayed, the secret code will be shown, and you can type it into your phone instead of scanning the QR code.

If the code is correct, your account will be linked to the app on your phone. When you sign in to Memsource again, you will be prompted to enter your username, password and then one of the codes in your phone app.

 

Note: Two-factor Authentication is no longer available for users who log in using a Google account as 2-Step Verification is already available for Google users. Learn more about 2-Step Verification in Google.
