New Features and Improvements

Introduction to Memsource Editor for Mobile

Memsource Editor for Mobile now features a brief introduction when you log in for the first time. This introduction is shown the first time a user logs in as a Project Manager and the first time they log in as a Linguist and is available for both Android and iOS phones.

 

Fixed Bugs

  • Opening large jobs would sometimes cause an error in loading the data. This has been resolved.
  • When swiping between segments with an Apple phone, the cursor would sometimes be placed at the end of the segment. This has now been changed so that the cursor is at the start of the segment.
  • When opening a job with an Android and then clicking to open the segment detail screen, the keyboard did not automatically open. This has now been fixed.
  • In addition, this release also includes general minor stability improvements for Android phones.
