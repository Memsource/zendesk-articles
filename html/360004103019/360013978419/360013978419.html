Click the username and/or user image from the top right corner to view user profile options. When logged in, the user can see what role they have in Memsource, update their profile and see their Login History (dependent on user role) or Sign Out from Memsource.

To update a user profile, follow these steps:

  1. Click the the username and/or user image from the top right corner.

    The Profile Setup menu is displayed.

  2. Click My Profile.

    The My Profile page opens.

  3. Update Profile Picture and/or User Details and/or Password.

  4. Click Update Profile Settings.

    Change are saved to the user profile.
