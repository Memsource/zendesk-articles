New Features and Improvements

New Icon for CAT Pane

Previously, a plus icon (+) was used to navigate to the CAT Pane. This has now been replaced with a Menu icon.

 

Segment Numbers Now Shown

Segment numbers are now shown for each segment in a job, and the total number of segments in a job is also shown. The panel containing the total number of segments will be hidden when a user scrolls down and will be displayed again when scrolling up. 

 

Inserting Tags Button has Moved

The button to insert tags has been moved to the Tools section.
